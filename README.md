# 余震 · DROP — 第一人称地震逃生科普游戏

> 黑客松项目（4 人 / 2 天 / 8MB 纯前端）。
> **你是地震中的普通人，第一人称。凭本能做的"逃命"选择往往是致命的——你死了，镜头拉出来告诉你为什么，再来一次。** 用恐惧把救命知识焊进脑子。

## 🎮 快速开始

clone 下来后，在项目根起一个**静态服务器**（任选其一），不能直接双击 `index.html`（用了 ES module + CDN，需要 http 环境）：

```bash
# 方式一：Python（macOS/Linux 自带）
python3 -m http.server 8137
# 然后浏览器打开 http://localhost:8137/

# 方式二：Node
npx serve .

# 方式三：VSCode 装 "Live Server" 插件，右键 index.html → Open with Live Server
```

**建议 Chrome + 戴耳机**（声音是这个游戏的灵魂）。

## 怎么玩

- **WASD** 走动，**鼠标** 环顾四周，**ESC** 松开鼠标
- 地震来时，**走到你认为安全的地方按 E 躲避**
- 选错 → 「你死了」→ 镜头拉出揭示原因 → 再来一次 → 爬更高的楼（越高越难）

## 🛠 技术栈

- **纯前端单文件** `index.html`，零构建、零依赖安装
- **Three.js**（CDN 引入）渲染 3D；**WebAudio** 程序化合成全部音效
- **几何体全部代码生成，零图片零音频文件**——守住 8MB 打包红线
