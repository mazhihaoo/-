============================================

📦 使用方法（按你的设备选择）：

┌─ 💻 Windows 电脑 ──────────────────────┐
│                                         │
│  1. 解压整个文件夹                       │
│  2. 双击 "启动服务器.bat"                │
│  3. 浏览器自动跳转或手动打开:            │
│     http://127.0.0.1:8137               │
│                                         │
│  ⚠️ 不要直接双击 index.html！            │
│    必须通过启动脚本运行                   │

┌─ 🍎 Mac 电脑 ──────────────────────────┐
│                                         │

│  3. 浏览器打开:                          │
│     http://127.0.0.1:8137               │
│                                         │
│  如提示安全问题，右键→打开即可            │

🎯 游戏简介：
  地震后的建筑废墟中，寻找并营救被困的孩子，
  管理体力与血量，避开危险区域，安全逃生！

🕹️ 操作：
  WASD = 移动
  鼠标 = 视角
  Shift = 跑步
  Ctrl = 蹲下
  E = 交互（开门/拾取/救人）
  F = 手电筒
